Coping Methods App

The Coping Methods App is an interactive Shiny application included with this package. It enables users to explore mental-health–related data through visualisations such as:

  • depression trends across age groups
  • year-to-year changes
  • comparison of coping strategies between countries

Because the app runs locally, this page provides static screenshots, key insights, and clear explanations to help readers understand the functionality before launching it.

To run the app:

library(copingmethods)
launch_copingmethods()

1. Depression by Age — Country View

What this visualisation shows

This tab displays depression percentages across age groups for a selected country/entity and year.

The screenshot below shows Afghanistan (1990).
The bar chart visualises depression levels across age groups, while the table on the right provides:

  • depression (%) for each age group
  • differences relative to the comparison year

Figure 1: Depression percentages by age group for Afghanistan in 1990.

Key insights

Tip
  • Depression increases steadily across age groups in the displayed dataset.
  • The table provides precise numerical values alongside the visual chart.
  • This view makes it easy to compare age-specific depression profiles across countries and years.

2. Year-to-Year Comparison (Trend Analysis)

What this visualisation shows

This tab compares how depression percentages have changed over time relative to a baseline year.

The screenshot below shows America (IHME GBD), 2009, compared against 1990.

The plot includes:

  • depression (%) by age group in 2009
  • a yellow dashed reference line showing the world average depression level
  • a table that lists:
    • depression in 1990
    • depression in 2009
    • the percentage differences

Figure 2: Depression percentages in America (IHME GBD) for 2009 compared with 1990.

Key insights

Tip
  • Several age groups show increases or decreases between 1990 and 2009.
  • The “Standardised” row provides an overall summary measure of change.
  • The global average line allows quick comparison between America and global levels.

3. Coping Methods Comparison — Between Countries

What this visualisation shows

This tab compares coping strategies used in two selected countries.

The screenshot below compares Australia and India.
The chart illustrates coping strategies such as:

  • talking to friends or family
  • lifestyle improvements
  • spending time in nature
  • seeking professional help
  • medication use
  • work or social adjustments

Figure 3: Comparison of coping methods between Australia and India.

Key insights

Tip
  • Australia shows greater use of professional services and lifestyle-based coping methods.
  • India shows stronger use of community and family-based support strategies.
  • The ranked bar chart clearly highlights cultural differences in coping preferences.

Why This Page Exists

Because hosting Shiny apps online requires paid servers,
this documentation page ensures the app is still fully understandable by providing:

  • static screenshots
  • clear explanations
  • highlighted insights
  • a faithful visual representation of the interface

This allows assessors, collaborators, and website viewers to understand the app without needing to run it immediately.
